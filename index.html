如何使用windows terminal打开git bash
发表于2023-03-31更新于2023-07-31 环境配置
今日，搭建博客的时候，使用git进行管理，但是新版的git bash的界面太丑了 于是想要在windows terminal中使用git bash 步骤如下： 1.搜索“终端” 2.点击“设置” 3.在setting.json文件中的profiles的list中添加以下内容 { "guid" : "{5D1F95DF-36E8-56AD-C203-EA75CE06422C}", "name" : "Git Bash", "commandline" : "E:\\Git\\bin\\bash.exe --login -i",}, 其中commandline这一项填写的是自己的Git的安装路径（bin） 参考链接：https://www.cnblogs.com/cong-wang/p/15026535.html
