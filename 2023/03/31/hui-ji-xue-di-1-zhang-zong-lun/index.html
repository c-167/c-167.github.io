会计学-第1章-总论

一、会计的基本概念

1.1会计的产生和发展

1.1.1会计的产生

会计起源的标志：

  • 中国——书契（帐单）
  • 古巴比伦——黏土记录板
  • 印度——贝多罗树叶记录
  • 埃及——纸草文书

会计产生的根本原因：人们对经济效益的追求（以尽可能少的劳动消耗，尽可能节省的劳动费用，获得尽可能大的劳动成果）。

“会计”一词的起源：根据郭道扬教授的说法，“会计”命名起源于西周。

  • 西周“官厅会计”核算的具体情况考察,“会计”二字连用的基本含义在当时是:既有日常零星核算,又有岁末的总合核算,通过日积月累到岁末的核算,达到正确考核王朝财政经济收支的目的。

1.1.2会计的发展

  • 知识点：
    • 问：会计的原始含义是？
      • 答：记账、算账、报账
  • 1.古代会计（由单式簿记演进为复式簿记的萌芽）
    • 唐宋两代的“四柱清账”
      • 相当于现在的：上期结存+本期增加=本期减少 +本期结存
    • 明末清初的“龙门账”
      • “进－缴＝存－该”
    • 印度原始公社的独立记账员
  • 2.近代会计（运用复式簿记开始到20世纪50年代）
    • 从古代会计到近代会计的标志性事件有两个：
      • 🎯1494年,意大利数学家卢卡·帕乔利所著《算术、几何、比及比例概要》一书问世。（系统地介绍了复式记账法）
      • 🎯1854年世界上第一个会计师协会——英国爱丁堡会计师公会的成立
    • 近代会计实现的两个根本转变：
      • 单式簿记-->复式簿记
      • 簿记时代-->会计时代
  • 3.现代会计（20世纪50年代后）
    • 形成了会计的两大分支（适应企业管理的需要）
      • 财务会计——对外提供企业经营信息
      • 管理会计——队内提供管理决策信息
    • 并且设立了审计环节
    • 面临着新的挑战
  • 知识点：
    • 问：会计的原始含义是？
      • 答：记账、算账、报账

1.2会计环境

会计环境的含义：是指影响并决定着会计的产生与发展的社会客观条件和特殊情况，包括：会计思想、会计法制、会计理论、会计方法等及与其密切相关的审计等。

  • 会计理论：比如会计信息质量要求、会计假设、会计职能、会计对象、会计目标

    • 会计的目标：获取信息🚩

  • 会计方法：包括单式记账法，复式记账法等

  • 审计：包括内部审计、政府审计、民间审计

1.3会计的含义

  • 原始的会计含义：
    • “零星算之为计，总合算之为会”
    • 会计的原始含义就是记账、算账、报账
  • 现代会计的含义：
    • 信息系统论
    • 管理活动论
    • 管理工具论
    • 艺术论

我们的观点：

会计是以货币作为主要计量单位，运用专门的方法，对企业单位发生的经济活动进行确认、计量、记录和报告，向信息使用者提供连续、系统、完整的会计信息。

会计的特点：

  • 1.以货币为主要计量单位
  • 2.以凭证为基本依据
  • 3.以提供连续、系统、完整的财务信息为己任

1.4会计分类

1.4.1按报告对象不同的划分

  • 1.财务会计
    • 向企业外部的信息使用者提供决策所需的会计信息
  • 2.管理会计
    • 管理会计是为企业内部管理者，尤其是高层管理者提供控制、预测、决策所需的信息

1.4.2按会计目的的不同划分

  • 1.营利组织会计（企业会计）
  • 2.非营利组织会计（政府会计）

注意：“营利”和“盈利”的区别：

  • 营利是按照活动的目的划分的
  • “盈利”是指单位组织的总收入大于总支出的差额，它反映的是结果而不是目的

1.4.3按对象的范围来划分

  • 1.宏观会计
    • 宏观会计是以整个社会经济活动作为会计的对象
  • 2.微观会计
    • 微观会计则以会计个体作为会计对象
    • 之前提到的财务会计、管理会计以及营利组织会计和非营利组织会计均属于微观会计

1.5会计目标

用4个字来概括会计的目标是什么，“提供信息”

  • 这里的信息指的就是4张报表
    • 资产负债表
    • 利润表
    • 现金流量表
    • 所有者权益表

会计的基本目标：向会计信息使用者提供决策有用信息和反映企业管理层受托责任履行情况。

  • 决策有用观下的会计目标
  • 受托责任观下的会计目标

具体的会计目标：

  1. 企业的财务状况及其变化
  2. 企业的经营业绩
  3. 现金流量
  4. 所有者权益

二、财务会计

2.1财务会计的含义与特点

财务会计是现代会计的一个重要分支，与管理会计一起共同服务于市场经济条件下的现代企业

财务会计的特点：

  1. 财务会计主要提供的是财务信息
  2. 财务会计全面、连续、系统和综合地提供有关企业已经发生的经济活动或事项的财务信息
  3. 财务会计必须按统一的会计准则要求提供财务信息
  4. 财务会计有一套完整、统一的会计处理程序和方法

2.2财务会计对象

2.2.1财务会计的对象的界定

财务会计对象就是指财务会计核算的内容，即财务会计的客体。

知识点：

  • 问：会计对象是指会计工作的（）A.单位 B.企业 C.公司 D.客体
    • D.客体

目前主要的意见是将财务会计对象定位于“资金运动论”。

  • 即财务会计的对象就是社会再生产过程中的资金运动
  • 包括资金的投入、资金的运用（资金的循环与周转）、资金的退出
  • 资金的退出
    • 资金的退出包括偿还各项债务、缴纳各项税金、返回投资者及向所有者分配股利或利润等内容。
    • 提取盈余公积属于会计学中的资金退出。在会计中，资金退出是指企业将部分或全部已经投入的资金收回或转移走的过程，包括提取储备基金、提取盈余公积、分红派息等。提取盈余公积属于一种常见的资金退出方式，其目的是为了满足企业的发展需要或者进行股东分红等活动。
      • 提取盈余公积：提取盈余公积是指企业利用其留存的盈余公积金，将部分或全部金额转入资本账户，增加公司的注册资本或用于向股东派发红利。盈余公积金是企业留存的一部分利润，在一定限制条件下可以进行提取，包括法定盈余公积金和任意盈余公积金。企业需要遵守相关的法律法规，依照规定的程序和比例提取盈余公积，并及时履行披露义务，对外公布提取盈余公积的情况。
    • 向债权人支付利息不属于资金退出。因为企业向债权人支付利息并不是将部分或全部已经投入的资金收回或转移走的过程。相反，这是企业向借款人支付应付的债务利息，即企业向债权人支付的一种报酬。在会计上，企业需要按照实际发生的情况记录和核算利息支出，并及时进行披露。
  • 资金的投入
    • 资金的投入包括所有人投入和向债权人借入的资金。前者构成企业所有者权益，后者形成企业的债权人权益，即企业的负债。
  • 资金的运用
    • 可划分为供应过程、生产过程、销售过程三个阶段
    • 商业资金的运动过程：货币资金——商品资金——货币资金
    • 工业资金的运动过程：货币资金——储备资金——（发生了折旧）生成资金——产成品资金——货币资金

2.3财务会计的职能

财务会计具有反映、控制、评价、预测和决策5项职能。

其中反映和控制是财务会计的基本职能。

反映是决定会计本质的首要职能。

反映又称为核算职能，是指以货币为主要计量单位，通过确认、计量、记录、计算、报告等环节，提供特定主体经营活动的财务信息的功能。

控制又称为监督职能，是指在进行会计核算的同时必须对特定主体的合法性和合理性进行严格审查。

2.4财务会计方法

财务会计方法：包括核算方法、分析方法和检查方法。其中会计核算方法是基本方法。

核算方法主要包括以下几个方法：

  • 设置账户
  • 复式记账
  • 填制和审核凭证
  • 登记账簿
  • 成本计算
  • 财产清查
  • 编制财务报表

  • 其中初始环节又叫做“做账”
  • 中间环节又叫做“登账”
  • 最终环节又叫做“编报表”

口诀：做账、登账、编报表
